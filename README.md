# Gelişmiş Modern Portföy

Bu proje, React ve Vite kullanılarak hazırlanmış sade ve modern bir portföy sitesi şablonudur. Site, kişisel tanıtım, projeler, iletişim ve diğer önemli bölümler için temel bir yapı sunar. Proje, genel bir portföy şablonu olarak tasarlandı; kaynak kodu açık olarak paylaşılabilir ve üzerinde kolayca düzenleme yapılabilir.

## Ne İçerir?

- React bileşenleri ile oluşturulmuş hızlı ve düzenli bir sayfa yapısı
- Vite ile geliştirme ve dağıtım için optimize edilmiş yapı
- Anasayfa, Hakkımda, Projeler, Proje Detayları ve İletişim sayfaları
- Proje kartları ve teknik etiketler için sade bileşenler
- Küçük ve kolayca özelleştirilebilir CSS stili

## Kurulum

1. Depoyu klonlayın veya zip olarak indirin
2. Proje dizinine gidin
3. Gerekli paketleri yükleyin:

```bash
npm install
```

## Geliştirme Sunucusu

Aşağıdaki komutla yerel geliştirme sunucusunu başlatabilirsiniz:

```bash
npm run dev
```

Ardından tarayıcınızda `http://localhost:5173` adresini açın.

## Proje Paylaşımı ve Açık Kaynak

Bu proje GitHub üzerinde açık bir depo olarak paylaşılmak için uygundur. Projeyi herkese açık hale getirip README dosyasını güncel tutarak diğer geliştiricilerin ve ziyaretçilerin projeyi incelemesini sağlayabilirsiniz.

> Not: Projeyi paylaşmadan önce kişisel içerikleri (e-posta, telefon numarası vb.) gerektiği şekilde kontrol edin.

## Özelleştirme

- `src/data/siteContent.js` dosyasından içerik, metin ve proje bilgilerini düzenleyebilirsiniz.
- `src/components/` klasöründeki bileşenleri kendi tasarımınıza göre uyarlayabilirsiniz.
- `src/index.css` dosyasında renk, yazı tipi ve düzenlemeleri değiştirebilirsiniz.
